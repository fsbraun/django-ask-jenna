==================
django-ask-jenna
==================

Adds generative AI support to Django admin and Django CMS.

**django-ask-jenna** consists of two main packages:

* **ask_jenna**: A reusable Django app for AI-powered prompt management with an autodiscovery system
* **cms_mcp**: A Model Context Protocol (MCP) server implementation that exposes Django CMS content and operations to AI tools
